Paso 2 - Calcular el campo magnético B₁

En este paso analizaremos únicamente el campo magnético generado por el conductor I₁ sobre el punto P₁.

{{ grafico | safe }}
Paso 1 - Calcular el vector radio r₁

El vector radio une el conductor I₁ con el punto de observación P₁.

  • Δx = {{ dx1 }} cm
  • Δy = {{ dy1 }} cm

r₁ = √(Δx² + Δy²)

r₁ = √({{ dx1 }}² + {{ dy1 }}²)

Resultado

r1 = {{ "%.2f"|format(r1) }} cm

Paso 2 - Calcular el módulo de B₁

Aplicamos la expresión del campo magnético producido por un conductor rectilíneo infinito. 


B1 = μ₀ · I1
    ───────
     2π · r1

I1 = 1 A

r1 = {{ "%.4f"|format(r1 / 100) }} m

Resultado

|B1| = {{ "%.2e"|format(b1) }} T

Paso 3 - Determinar la dirección

Aplicando la regla de la mano derecha, el vector B₁ es perpendicular al vector radio r₁ y apunta en el sentido mostrado en el gráfico.
